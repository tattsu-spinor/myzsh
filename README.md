## 初期設定
1. ~/.zshrcを削除する。(必要ならバックアップを取る。)
2. このレポジトリを任意の場所にgit cloneする。
3. レポジトリのルートディレクトリに`cd`する。
4. `chmod +x init.sh`を実行して実行権限を付与。
5. `./init.sh`を実行する。
6. ターミナルの再起動 or `source ~/.zshrc`を実行すれば完了。
